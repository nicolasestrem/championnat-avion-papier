## Comment utiliser le gabarit

1. **Imprimez** le PDF à 100 % (sans « ajuster à la page »), afin que les proportions et les repères restent exacts.
2. **Découpez** la feuille le long du cadre extérieur si le modèle le demande.
3. **Pliez** en suivant les lignes : les traits pleins indiquent les plis principaux, les pointillés les plis secondaires.
4. **Ajustez** les ailes bien symétriques, puis lancez d'un geste souple et légèrement vers le haut.
